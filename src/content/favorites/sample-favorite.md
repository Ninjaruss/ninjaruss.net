This is a sample favorite entry to demonstrate the new favorites gallery feature.

You can add your favorite characters, anime, manga, films, games, and more here. Each favorite can have:

- A title
- An emblem image (Yu-Gi-Oh style card)
- Tags for organization
- A type (character, anime, manga, film, music, book, game, other)
- Markdown content describing why it inspires you
- Collections for cross-referencing with other content

## How to use

1. Create a new markdown file in `src/content/favorites/`
2. Add the required frontmatter fields
3. Write your content
4. The gallery will automatically include it!

## Example frontmatter

```yaml
---
title: "Your Favorite Thing"
tags: ["tag1", "tag2"]
collections: ["collection name"]
publishedAt: 2026-01-10
emblem: "/images/emblems/your-emblem.svg"
favorites_type: "anime"
---
```
